Начало
В начале сотворил Бог небо и землю, а точнее небольшое окно регистрации с 2-мя полями и 1-ой кнопкой. 1-ое поле – Логин, 2-ое – Пароль. Кнопка – Зарегистрироваться.
Функции:
Пользователь может зарегистрироваться.
Ограничения:
•	Логин нового пользователя должен состоять из 6 – 20 символов включительно.
•	Не может содержать символ «.» и символ «*».
•	Пароль должен состоять из 3 – 7 символов.
•	Не может совпадать с полем Логин.
•	Не может содержать пробел.

Тестовые сценарии
Предусловие:
Открыть окно регистрации.
Сценарий 1(Гуд): пользователь регистрируется с правильными данными.
Ввести в поля следующие данные: Логин = Тираннозавр, Пароль = Есть.
Нажать кнопку Зарегистрироваться.
1.	Проверить, что пользователь появился в базе данных.
2.	Проверить, что данные появившиеся в базе данных совпадают с введенными.

Тру Тест – Кейсы
Позитивный
Действие	Ожидаемый результат
1.Открываем форму регистрации.	•	Форма открыта.
•	Все поля по умолчанию пусты.
•	Кнопка Зарегистрироваться активна.
2.Заполняем поля формы
•	Логин = Тираннозавр 
•	Пароль = Есть	•	Поля заполнены.
3.Нажимаем кнопку отправить.	•	Сообщение «Вы успешно зарегистрировались» выведено на экран.
•	Пользователь появился в базе данных.



Негативный
Действие	Ожидаемый результат
1.Открываем форму регистрации.	•	Форма открыта.
•	Все поля по умолчанию пусты.
•	Кнопка Зарегистрироваться активна.
2.Заполняем поля формы
•	Логин = Про.100  
•	Пароль = Есть	•	Поля заполнены.
3.Нажимаем кнопку отправить.	•	Валидационное сообщение с ошибкой выведено на экран:
«В поле Логин запрещено использование спец. Символа «.»».
•	Пользователь не появился в базе данных.
